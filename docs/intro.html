[N2M] 수자원 시설 현황 3D 지도 개발 (2021.03 ~ 현재)

수자원 시설물과 실제 디바이스를 연동한 3D 지도 프로젝트

프로젝트 상세

  • 외부 디바이스와 데이터 연동하여 현재 프로젝트와 내부 오브젝트 간 연결
  • 연동된 기기 데이터를 기반으로 지도 상에 정보를 표시하고 각 오브젝트별 데이터 매핑
  • 주요 스택: Spring Boot, Geoserver, PostGIS, PostgreSQL, Vue.js, Cesum.js
  • 개발 인원: 2명

역할 및 성과

  • [Spring Boot] Shapefile 위치정보 파싱 후 DB 적재
  • [Spring Boot] Mybatis(@SelectProvider, @UpdateProvider)를 활용한 DDL 작성
  • [Spring Boot] Service - OkHttpClient - 외부 Geoserver API 연동
  • [Spring Boot] OAuth 2.0 기반 토큰 유효성 및 인증 처리
  • 지도 상에 표시할 정보를 나타낼 오브젝트 개발(glTF, shapefile, GeoJSON)
  • WFS 기반으로 지도상에 데이터 표시
  • glTF 및 GeoJSON 에디터 개발
  • PostGIS 위치 정보 데이터 관리 및 개발
  • 외부 디바이스와 서버 간 인터페이스 설계 및 기능 구현
  • Gitlab, SVN, Jenkins, Docker를 통한 배포 관리
  • Mattermost, Redmine 을 통한 협업
